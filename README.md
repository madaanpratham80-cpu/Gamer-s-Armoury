🛡️ Gamer's Armoury: The Performance Architect

**Gamer's Armoury** is a high-performance web dashboard designed for enthusiasts to bridge the gap between AAA gaming and hardware capabilities. Inspired by the **ASUS Armoury Crate** ecosystem, this application allows users to predict game performance based on specific graphics presets and hardware benchmarks.

 🚀 Core Features (Milestone 1 Planning)

* **Nexus Search Engine:** Integration of the **RAWG API** to fetch real-time system requirements for 500,000+ games. 
* **Preset Matchmaker:** A logic engine where users select a graphics preset (**Low, Medium, High, Ultra**) to see compatible hardware. 
* **The "User Rig" Profile:** A dedicated section to save and compare current hardware specs using **Local Storage**.
* **Armoury-UX Interface:** A dark-themed, responsive dashboard featuring neon accents, performance "Gauges," and a modular layout. 
* **Optimization Vault:** Leverages the **GitHub API** to find community-driven optimization mods and performance scripts for searched titles. 

🛠️ Technical Stack & Strategy

* **UI Development:** Developed using **HTML5** and **Tailwind CSS** to achieve a futuristic gaming aesthetic.
* **Logic Layer:** Built with **Vanilla JavaScript** (ES6+), focusing on the **Fetch API** for data integration. 
* **Data Processing (HOFs):** The project **strictly** utilizes Array Higher-Order Functions for all operations: 
    * `.filter()`: To isolate hardware that meets specific benchmark thresholds. 
    * `.sort()`: To rank laptops by price-to-performance ratios. 
    * `.map()`: To dynamically render game cards and hardware data. 
* **Evolutionary Path:** Initially developed in Vanilla JS for core logic mastery, with a planned refactor to **React** as the semester progresses.
