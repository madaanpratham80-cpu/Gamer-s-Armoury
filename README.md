# 🛡️ Gamer's Armoury: The Hardware Architect

**Gamer's Armoury** is a premium, high-performance web dashboard designed for enthusiasts to track, analyze, and compare gaming laptops. Inspired by the **ASUS Armoury Crate** ecosystem, this application allows users to build their hardware profile, monitor benchmark scores, and perform side-by-side spec comparisons to find the ultimate gaming machine.

## 🚀 Core Features

* **The System Profile:** A dedicated setup manager to configure, save, and visualize your current hardware specs (CPU, GPU, RAM, Storage) via **Local Storage**.
* **Head-to-Head Comparison:** A robust side-by-side comparison tool that breaks down hardware metrics, benchmark scores, and calculates price-to-performance (Value/Lakh) ratios to crown a winner.
* **Armoury-UX Interface:** A sleek, fully responsive dark-themed dashboard featuring modular layouts, animated progress bars, and premium micro-interactions.
* **Hardware Database:** A curated database of modern gaming laptops (ASUS, MSI, Lenovo, Acer, HP, Dell, Razer, etc.) loaded with detailed variants and relative benchmark scores.

## 🛠️ Technical Stack & Strategy

* **UI Development:** Crafted with **HTML5** and **Vanilla CSS**, leveraging modern structural design to achieve a premium, futuristic gaming aesthetic without reliance on heavy frameworks.
* **Logic Layer:** Built with **Vanilla JavaScript (ES6+)**, driving the interactive dropdowns, dynamic component rendering, and hardware comparisons.
* **Data Processing:** The project utilizes Array methods (`map`, `filter`, `sort`) to retrieve and render specific laptop variants, calculate overall scores, and parse user hardware selections seamlessly.

## ⚙️ How to Use

1. **Dashboard (Home):**
   - Click the **+** button to add your laptop. Select a brand/model and your exact hardware variant.
   - View your customized system usage gauges and benchmark scores across CPU, GPU, RAM, and Storage.
2. **Compare Tool:**
   - Navigate to the **VS Compare** tab in the sidebar.
   - Pick "Laptop A" and "Laptop B" from the comprehensive dropdown menus.
   - Click **Compare Now** to generate a detailed breakdown of specifications, relative bar charts, and a final verdict based on raw power and overall value.

## 📅 Project History

Gamer's Armoury began as a broader project exploring both hardware and gaming catalogs. As the project evolved, it was sharply refined into a specialized **hardware tracking and comparison utility** to provide gamers with a powerful, fast, and highly targeted application.
